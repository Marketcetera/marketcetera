Destination management.

The classes in this package effect destination management:

  • {@link org.marketcetera.ors.dest.SpringDestination} and {@link org.marketcetera.ors.dest.SpringDestinations} respectively provide Spring-based configuration of a single destination, and all destinations collectively.

  • {@link org.marketcetera.ors.dest.SpringSelectorEntry} and {@link org.marketcetera.ors.dest.SpringSelector} respectively provide Spring-based configuration of a selector entry, and the selector.

  • {@link org.marketcetera.ors.dest.Destination} and {@link org.marketcetera.ors.dest.Destinations} respectively represent in memory a single destination, and all destinations collectively.

  • {@link org.marketcetera.ors.dest.SelectorEntry} and {@link org.marketcetera.ors.dest.Selector} respectively represent in memory a selector entry, and the selector.
